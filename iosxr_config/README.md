<br><br>

## 留意事項

- XRv9000はSRv6を実装しています

- XRv9000はSRv6 uSIDを実装しています

- XRv9000はSRv6 TEを実装しています

- XRv9000でTE Policyを使うためにはuSIDが必要です

- CRS1000vはEVPNを実装しています

- CSR1000vはSRv6を実装していません
